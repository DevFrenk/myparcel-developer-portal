MyParcel provides a suite of APIs to power your shipping integration. Select an API below to view the full reference documentation.

## Available APIs

### [Shipment API →](/api/shipping)
The core MyParcel API. Create shipments, generate labels, track packages, manage delivery options, webhooks and notifications across multiple carriers.

**Base URL:** `https://api.myparcel.nl`

---

### [Order API →](/api/orders)
Manage the complete order lifecycle — from creation through packaging, shipment configuration, and status tracking. Supports bulk operations and 16+ carriers.

**Base URL:** `https://order.api.myparcel.nl`

---

### [IAM API →](/api/iam)
Identity & Access Management — verify your credentials, check permissions, and inspect feature flags for your account.

**Base URL:** `https://iam.api.myparcel.nl`

---

### [Rule API →](/api/rules)
Configure shipping rules for automatic shipment options based on custom conditions.

**Base URL:** `https://rule.api.myparcel.nl`

---

### [Address API →](/api/address)
Address validation and autocomplete for accurate delivery across supported countries.

**Base URL:** `https://address.api.myparcel.nl`

---

### [Printing API →](/api/printing)
Generate and manage shipping labels and documents for your shipments.

**Base URL:** `https://printing.api.myparcel.nl`

---

### [Product API →](/api/products)
Browse available products, carriers, and shipping options for your account.

**Base URL:** `https://product.api.myparcel.nl`
