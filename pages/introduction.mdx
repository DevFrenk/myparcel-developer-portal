Welcome to the MyParcel Developer Portal. Our APIs enable you to integrate shipping, order management, and identity services directly into your application.

## Getting Started

1. Create an account via the [MyParcel backoffice](https://www.myparcel.nl/)
2. Generate an API key under **Settings** → **Integration** ([direct link](https://backoffice.myparcel.nl/settings/integration/))
3. Check the available [SDKs and integrations](/documentation) if you're working with an existing platform

## APIs

| API | Description |
|-----|-------------|
| [Shipment API](/api/shipping) | Create shipments, generate labels, track packages and delivery options |
| [Order API](/api/orders) | Full order lifecycle — from creation to shipping and status tracking |
| [IAM API](/api/iam) | Identity & Access Management — verify credentials and manage permissions |
| [Rule API](/api/rules) | Configure shipping rules for automatic shipment options |
| [Address API](/api/address) | Address validation and autocomplete for accurate delivery |
| [Printing API](/api/printing) | Generate shipping labels and documents |
| [Product API](/api/products) | Browse available products, carriers and options |
