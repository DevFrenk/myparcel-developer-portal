Welkom bij de MyParcel Developer Portal. Onze API's stellen je in staat om verzending, orderbeheer en identiteitsdiensten direct in je applicatie te integreren.

## Aan de slag

1. Maak een account aan via de [MyParcel backoffice](https://www.myparcel.nl/)
2. Genereer een API key via **Instellingen** → **Integratie** ([directe link](https://backoffice.myparcel.nl/settings/integration/))
3. Bekijk de beschikbare [SDK's](/documentation) als je met een bestaand platform werkt

## API's

| API | Beschrijving |
|-----|-------------|
| [Shipping API](/api/shipping) | Aanmaken van zendingen, labels genereren, tracking en bezorgopties |
| [Order API](/api/orders) | Volledige order lifecycle — van creatie tot verzending en statustracking |
| [IAM API](/api/iam) | Identity & Access Management — credentials verifiëren en permissies beheren |
| [Rule API](/api/rules) | Verzendregels configureren voor automatische shipment-opties |
| [Address API](/api/address) | Adresvalidatie en autocomplete voor nauwkeurige bezorging |
| [Printing API](/api/printing) | Verzendlabels en documenten genereren |
| [Product API](/api/products) | Beschikbare producten, carriers en opties bekijken |

## CMS Integraties

Browse de beschikbare platformen op de documentatiepagina's in het menu. Vragen over plugins kun je stellen via [support@myparcel.nl](mailto:support@myparcel.nl).
