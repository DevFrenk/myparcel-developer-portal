Welcome to the MyParcel API documentation. Our APIs enable you to integrate shipping, order management, and identity services directly into your application.

## APIs

### Shipping API

The core MyParcel API for creating shipments, generating labels, tracking packages, and managing delivery options across multiple carriers.

[Go to Shipping API →](/api/shipping)

### Order API

Manage the complete order lifecycle — from creation through packaging, shipment configuration, and status tracking. Supports bulk operations and 16+ carriers.

[Go to Order API →](/api/orders)

### IAM API

Identity and Access Management — verify your credentials, check permissions, and inspect feature flags for your account.

[Go to IAM API →](/api/iam)

### Rule API

Configureer en beheer verzendregels voor automatische shipment-opties.

[Go to Rule API →](/api/rules)

### Address API

Valideer en autocomplete adressen voor nauwkeurige bezorging.

[Go to Address API →](/api/address)

### Printing API

Genereer en beheer verzendlabels en documenten.

[Go to Printing API →](/api/printing)

### Product API

Bekijk beschikbare producten, carriers en opties.

[Go to Product API →](/api/products)
